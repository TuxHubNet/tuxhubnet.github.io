Web Browsers

Firefox - Popular open-source browser with strong privacy features and extensive extension support.
Chrome - Fast and widely used browser developed by Google, known for performance and vast extension library.
Chromium - Open-source version of Chrome, offering similar features without proprietary Google components.
Brave - Privacy-focused browser with built-in ad-blocking and tracking protection; also supports earning rewards through its ad system.
Opera - Feature-rich browser with built-in VPN, ad-blocker, and social media integrations.
Vivaldi - Highly customizable browser aimed at power users, developed by former Opera team members.
Falkon - Lightweight browser developed for KDE, with a focus on speed and simplicity.
Epiphany (GNOME Web) - GNOME's default browser, designed to integrate smoothly with the GNOME desktop environment.
Edge - Microsoft’s Chromium-based browser with features for productivity, such as Collections and integration with Microsoft services.
Midori - Lightweight browser designed for speed and efficiency, using WebKit as its rendering engine.
Waterfox - Privacy-focused Firefox fork with support for legacy Firefox extensions.
Konqueror - KDE browser and file manager that offers unique functionality for KDE environments.
SlimBrowser - Lightweight browser focused on speed and customization, with built-in ad-blocking.
SeaMonkey - All-in-one suite with browser, email, and HTML editing capabilities, designed as a successor to the old Mozilla Suite.
Links - Text-based browser for users who prefer or need to browse without graphical support.
Lynx - One of the oldest text-based browsers, popular in terminal environments and accessibility settings.
IceCat - GNU version of Firefox, with enhanced privacy protections and no proprietary binaries.
Dooble - Privacy-focused, open-source browser with minimalistic design.
Surf - Minimalist web browser from the suckless project, designed to be simple and efficient.
Dillo - Extremely lightweight and fast browser, suitable for very low-resource systems.
Galeon - GNOME-based browser built for speed, though no longer actively developed.
Basilisk - Firefox fork that focuses on the traditional Firefox UI and XUL extensions.
NetSurf - Lightweight web browser with a custom rendering engine; good for low-spec systems.
Arora - Lightweight, cross-platform web browser with a focus on simplicity.
Iridium - Chromium-based browser focused on enhanced privacy and security settings.
Conkeror - Emacs-inspired, keyboard-driven browser for power users, focused on extensibility.
Uzbl - Minimalist, scriptable browser that focuses on Unix philosophy and user customization.
Qute - Keyboard-driven browser with a minimalist interface, inspired by Vim.
Blisk - Developer-focused browser with tools for responsive design and web development.
Pale Moon - Firefox-based browser with a traditional interface, designed to be lightweight and customizable.
LibreWolf - Firefox fork with privacy and security enhancements, no telemetry, and enhanced privacy settings.
Tor Browser - Privacy-focused browser that uses the Tor network to anonymize browsing and protect against tracking.
Min Browser - A minimalist browser designed for simplicity, fast browsing, and a distraction-free experience.
Netsurf - Lightweight web browser with a small footprint; ideal for low-spec systems or minimal setups.
Otter Browser - Aiming to recreate the best aspects of the classic Opera 12 interface with modern security.
Nyxt - Keyboard-centric, Lisp-scriptable browser designed for power users.
Next Browser - Emacs-inspired, keyboard-focused web browser geared toward power users.
Luakit - Lightweight, highly configurable browser based on WebKit, with a Lua scripting interface.
Viper Browser - Lightweight, open-source browser with a focus on privacy and minimalism.
LibreFox - Privacy-focused fork of Firefox that includes various tweaks for enhanced security.
Xombrero - Security-focused, minimalistic browser with a simplistic interface and focus on privacy.
Bromite - Chromium-based with an emphasis on privacy, ad-blocking, and de-Googling. (Primarily Android, experimental Linux builds)
GNOME Web - GNOME’s default web browser, now with better support for modern web standards.
Rekonq - Lightweight browser developed for KDE, with a simple interface. (Deprecated, but still functional)
